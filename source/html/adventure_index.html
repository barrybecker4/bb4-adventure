Adventure

This project started as framework for building simple text based adventures. The scenes were stored in an xml file. I thought it would be cool to have images and sounds associated with the scenes, so I added a UI. When editing the xml file got tedious, I decided to add the graphical editor. You can edit the story if you know the password. Next I would like to add support for keeping track of found items and hit points. This story is based on a adventure in Dragon Magazine back in 1980. This framework can also be used for educational purposes.

I used to deploy this application as applets or using WebStart, but that fell out of favor, so now I use a zip file that you can unpack and run locally. After unpacking, go into the bin directory and run one of the following. Note that if you are on Windows, you need to use the *.bat version of the script to run.


Text Adventure

 bb4-text-adventure 'ludlow/ludlowScript.xml'

Graphical Adventure

 bb4-adventure 'ludlow/ludlowScript.xml'

Binary Tutorial (text version)

bb4-text-adventure 'learnBinary/binaryScript.xml'

Binary Tutorial (graphical version)

 bb4-adventure 'learnBinary/binaryScript.xml'
